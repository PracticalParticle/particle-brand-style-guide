# Particle Crypto Security LTD - Style Guide

Welcome to the Particle Crypto Security LTD Design System and Style Guide.

## Overview

This design system provides a comprehensive set of components, design tokens, and guidelines for building enterprise security applications. It's specifically designed for **BloxChain Protocol** - a compliance framework for regulated smart accounts.

## Design Principles

### Enterprise Security Focus
- Strong visual hierarchy
- Clear, unambiguous UI elements
- Security-first color choices
- High contrast for accessibility

### Trust & Professionalism
- Clean, minimalistic design
- Consistent spacing and typography
- Professional color palette
- Geometric triangle motifs

### Blockchain Technology
- Modern, tech-forward aesthetic
- Subtle references to blockchain concepts
- Data visualization ready components

## Getting Started

### Installation

```bash
npm install
```

### Usage

```tsx
import { Button } from '@/components/Button'

function App() {
  return <Button variant="primary">Click me</Button>
}
```

## Features

- 🎨 Enterprise Blue Theme
- 🌓 Dark/Light Mode Support
- 🔷 Geometric Triangle Patterns
- ♿ WCAG 2.1 AA Compliant
- 📦 Fully Typed with TypeScript
- 🎯 Tailwind CSS Integration

## Components

Browse the component library using the sidebar. Each component includes:
- Interactive examples
- Props documentation
- Usage guidelines
- Code snippets

## Design Tokens

All design tokens are available as TypeScript exports:

```tsx
import { colors, typography, spacing } from '@/tokens'
```

## Theme Support

The design system supports both light and dark themes. Use the theme toggle in the toolbar to switch between modes.
